# Brief: Система сброса пароля

**Related Issue:** [GitHub Issue #2](https://github.com/OlegPhenomenon/video-chat-and-translator/issues/2)
**Status:** active
**Date:** 03.04.2026

### Цель
Сейчас если пользователь забыл пароль, у него нет возможности восстановить доступ к аккаунту. Он полностью заблокирован.

### Для кого
Для зарегистрированных пользователей, которые пользуются нашим сервисом для перевода и озвучки видео, а также для общения с ИИ-чатом о контенте данного видео.

### Технический контекст
Это вторая задача проекта. На данном этапе система авторизации должна быть реализована. В качестве библиотеки используется гем `devise`.

### Желаемый результат
Зарегистрированные пользователи с подтверждённой почтой могут восстановить доступ к аккаунту при утере пароля. Функция доступна только пользователям с подтверждённым email. Для неподтверждённых пользователей восстановление пароля недоступно.
