# Brief: Профиль пользователя

**Related Issue:** [GitHub Issue #3](https://github.com/OlegPhenomenon/video-chat-and-translator/issues/3)
**Status:** active
**Date:** 04.04.2026

### Цель
После входа в систему пользователь не может просматривать и обновлять свои учётные данные: **email** и **пароль**. В результате нельзя самостоятельно сменить скомпрометированный пароль, исправить неверный email или восстановить доступ к аккаунту после смены почты — это ведёт к обращениям в поддержку и блокирует нормальное пользование сервисом.

### Для кого
Для зарегистрированных пользователей, которые пользуются нашим сервисом для перевода и озвучки видео, а также для общения с ИИ-чатом о контенте данного видео.

### Технический контекст
Система авторизации реализована с использованием гема Devise (фича [001-auth-email](../001-auth-email/brief.md)).

### Желаемый результат
Пользователь с подтверждённой почтой может самостоятельно просматривать и обновлять свои учётные данные (email, пароль) без обращения в поддержку.
