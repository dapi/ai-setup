# Brief: Система авторизации и подтверждения Email

**Related Issue:** [GitHub Issue #1](https://github.com/OlegPhenomenon/video-chat-and-translator/issues/1)
**Status:** active
**Date:** 03.04.2026

### Цель
Сейчас приложение открыто для всех без аутентификации. Любой может получить доступ к функциональности без идентификации. Нужен механизм, гарантирующий что каждый пользователь — реальный человек с валидным email-адресом.

### Для кого
Любой человек, который хочет использовать переводом видео и пообщаться с ИИ о его контенте. Приходит из поиска / по прямой ссылке. Для использования сервиса должен зарегистрироваться и подтвердить, что email принадлежит ему.

### Технический контекст
Это первая задача проекта. Без системы аутентификации невозможно реализовать ни одну пользовательскую функцию (видеочат, профиль, историю). Авторизация — обязательный фундамент для всего остального функционала.

У нас инициализирован проект на Ruby on Rails, база данных Postgresql. Девелопмент среда работает через докер.

### Желаемый результат
Только пользователи с подтверждённым email-адресом имеют доступ к функциональности приложения. Неавторизованные пользователи перенаправляются на страницу входа/регистрации.

Неподтверждённый пользователь не имеет доступа ни к одной функции приложения кроме страниц регистрации, входа и подтверждения email.
