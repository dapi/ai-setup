# Персональный структурированный медиа-каталог в форме веб-приложения (media-lib)

## Цель проекта
Предоставить конечному пользователь возможность создать и поддерживать структурированный справочный каталог для индивидульного и/или коллективного пользования по произвольной тематике.

## Основные особенности
Каталог представляет собой древовидную структуры. Листьями дерева являются медиа файлы (в основном фото и видео) с набором произвольных метаданных, таких как название, краткое описание, и пр. Пользователь может настраивать список метаданных по своему усмотрению.

Ключевыми особенностями/требованиями являются:
- удобная и простая навигация
- быстрая и удобная возможность редактирования/добавления материала
- возможность расшарить доступ к материалу другим пользователям и/или группам пользователей

## Примеры использования
- Каталог моделей коллекционера-моделиста
- Каталог видов столярных соединений для столяра-любителя
