## Этап 2 — Генерация спецификации

Задача:
> взять brief и создать spec.md

Агент:
- изучил код
- сгенерировал полноценную спецификацию (~250 строк)

Ключевые решения:
- HTTP Basic Auth через Staff

- RBAC:
  - admin/manager → 200
  - staff → 403
- password_digest NOT NULL

- Unambiguous:
  - не указана роль в тестах
- Specific:
  - миграция не учитывает существующие данные
- Missing edge cases:
  - нет проверок на invalid credentials
